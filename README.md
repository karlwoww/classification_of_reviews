# Классификация отзывов IMDB с использованием нейронной сети

Простая реализация нейронной сети для бинарной классификации отзывов набора данных IMDB (положительные/отрицательные).

## 📋 Описание проекта

Проект демонстрирует построение и обучение нейронной сети на датасете IMDB для определения тональности отзывов. Модель достигает точности ~91% на тестовой выборке.

### Основные характеристики:
- **Датасет**: IMDB movie reviews (10,000 наиболее частотных слов)
- **Задача**: Бинарная классификация (положительный/отрицательный отзыв)
- **Архитектура**: Полносвязная нейронная сеть
- **Точность**: ~91% на тестовой выборке
